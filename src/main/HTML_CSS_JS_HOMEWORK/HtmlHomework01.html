Tool set

Environment variables

  • System variables
  • User variables
  • Path variable
  • More info: https://www.java.com/en/download/help/path.html

Java SDK

  1. jdk.java.com
  1. Downloads -> JDK 11.x.x
  2. Install
  3. Add the JDK path to system PATH environment variable
  4. Test the installation with command
  1. java -version

Apache Maven

  1. maven.apache.org
  1. Download the latest binary zip
  2. Unzip to a directory of you choice
  3. Add the path to the bin folder of the installation to PATH system variable
  4. Create and new env variable called JAVA_HOME pointing to the JDK path
  5. Test the Maven installation with the following command:
  1. mvn --version

IntelliJ IDEA

  1. jetbrains.com
  1. Download IntelliJ IDEA Community Edition
  2. Install

GIT

  1. git-scm.com
  1. Create a new user environment variable called HOME, pointing to your user’s home directory (C:\Users\username)
  2. Add the GIT install path to PATH environment variable
  3. Test the GIT installation with command
  4. git
  5. Configure GIT with commands
  6. git config --global user.name “your user”
  7. git config --global user.email “your email”
